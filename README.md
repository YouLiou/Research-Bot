# AI Web Explorer: Research Agent

This is a smart AI research agent built with **Google Gemini 2.0** and **Serper API**. 
It can think, search the web, and summarize the latest information for you.

## Features
- **Smart Reasoning**: The AI decides if it needs to search the web to answer your question.
- **Web Search**: Integrated with Serper API to get real-time data from Google.
- **Auto-Summarization**: Summarizes multiple search results into a concise answer.
